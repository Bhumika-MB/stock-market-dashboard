# 📈 Stock Market Prediction Dashboard

An interactive Stock Market Prediction Dashboard built using **Python**, **Streamlit**, **Plotly**, **Pandas**, **scikit-learn**, and **Yahoo Finance API**.

The application provides real-time stock analysis, technical indicators, machine learning-based price forecasting, and investment recommendations through an easy-to-use dashboard.

---

## 🚀 Features

- 📊 Interactive stock price charts
- 🕯️ Line Chart and Candlestick Chart
- 📈 20-Day Simple Moving Average (SMA)
- 📉 50-Day Simple Moving Average (SMA)
- 📈 20-Day Exponential Moving Average (EMA)
- 📊 Daily Returns
- 📉 Volatility Analysis
- 📍 Relative Strength Index (RSI)
- 📈 Market Trend Detection
- 🤖 30-Day Stock Price Prediction
- 📋 Linear Regression vs Support Vector Regression Comparison
- 🎯 AI Buy / Hold / Sell Recommendation
- 🛡️ Risk Analysis
- 🏢 Company Information
- 🌍 Support for US and Indian stocks
